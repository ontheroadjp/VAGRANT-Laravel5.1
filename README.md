# VAGRANT-Laravel 5.x（Latest version）

## Description

* Vagrant による Laravel 5.x (最新版)の実行環境
* 環境構築は puppet で

## Environment

### LAMP

* Ubuntu 12.04 LTS（32bit）
* Apache 2.4.16
* MySQL 5.5.46
* PHP 5.5.31

### Applications

* Laravel 5.x （Latest）   
	``http://localhost:8888``

### Tools

* phpMyAdmin  
	`` http://localhost:8888/phpmyadmin/``

## Apache

* ドキュメントルール ``/var/www/public``  
	``http://localhost:8888/``

## MySQL

* ユーザー名 ``root``
* パスワード なし

	``$ mysql -u root -p``
